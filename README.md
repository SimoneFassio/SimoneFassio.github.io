## üöÄ Getting Started

### Quick Setup
1. Replace placeholder content with your personal information
2. Update social media links in the contact section
3. Add your CV as `cv.pdf` in the root directory
4. Customize project links to point to your GitHub repositories

### Customization Guide

#### Personal Information
- Update the name and title in `index.html`
- Modify the hero description and about section
- Update contact information (email, phone, location)

#### Projects
- Replace the example projects with your own work
- Update project descriptions, technologies used, and links
- Add screenshots or demo links

#### Skills
- Adjust skill levels in the progress bars (CSS `width` property)
- Add or remove skill categories as needed
- Update technology names and proficiency levels

#### Social Links
- Update social media URLs in the contact section
- Add your LinkedIn, GitHub, Twitter profiles
- Replace the CV download link with your actual resume

#### Colors and Styling
- Modify CSS custom properties in `styles.css` for color scheme
- Adjust fonts, spacing, and layout as needed
- Customize animations and transitions
